📌 Student Management System (CLI - Python)
This is a simple Command Line Interface (CLI) based Student Management System developed in Python. The system allows users to manage student records by performing basic operations such as adding, updating, deleting, and viewing student information.
Each student record contains ID, Name, and Grade, and all data is stored in a CSV file to ensure persistence. The program follows Object-Oriented Programming (OOP) principles by using separate classes for student data and management operations.
The system also includes basic validation, such as ensuring that each student ID is unique, to maintain data consistency.

🚀 Features


Add new student records


Update existing student details


Delete student records


View all students in formatted output


Data stored in CSV file (File Handling)


Unique ID validation



🛠️ Technologies Used


Python


File Handling (CSV)


Object-Oriented Programming (Classes & Methods)



📖 Purpose
This project demonstrates the use of file I/O and OOP concepts in Python while building a practical and easy-to-use management system
